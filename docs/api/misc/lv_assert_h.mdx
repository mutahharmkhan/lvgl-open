### LV_ASSERT

```c
#define LV_ASSERT(expr) \
    do { \
        if(!(expr)) { \
            LV_LOG_ERROR("Asserted at expression: %s", #expr); \
            LV_ASSERT_HANDLER \
        } \
    } while(0)
```
