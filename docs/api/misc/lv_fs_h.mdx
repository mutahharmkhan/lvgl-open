### lv_fs_make_path_from_buffer

Create a special object from buffer/ memory address which looks like a file and can be passed as path to <ApiLink name="lv_fs_open" /> and other functions accepting a path.

For example 
```c
//Create a PNG file from t a buffer and use it
lv_fs_path_ex_t p;
lv_fs_make_path_from_buffer(&p, 'A', my_buf, my_buf_size, "png");
lv_image_set_src(image1, &p);
```

```c
void lv_fs_make_path_from_buffer(lv_fs_path_ex_t *path, char letter, const void *buf, uint32_t size, const char *ext)
```

<span className="sr-only">Parameters</span>

| Name | Type | Description |
|------|------|-------------|
| `path` | <ApiLink name="lv_fs_path_ex_t" display="lv_fs_path_ex_t *" /> | path to a lv_fs_path_ex object |
| `letter` | `char` | the identifier letter of the driver. E.g. <ApiLink name="LV_FS_MEMFS_LETTER" /> |
| `buf` | `const void *` | address of the memory buffer |
