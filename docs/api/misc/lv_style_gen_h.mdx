### lv_style_set_align

Set the alignment which tells from which point of the parent the X and Y coordinates should be interpreted. Possible values are: <ApiLink name="lv_align_t" display="LV_ALIGN_DEFAULT" />, `LV_ALIGN_TOP_LEFT/MID/RIGHT`, `LV_ALIGN_BOTTOM_LEFT/MID/RIGHT`, `LV_ALIGN_LEFT/RIGHT_MID`, <ApiLink name="lv_align_t" display="LV_ALIGN_CENTER" />. <ApiLink name="lv_align_t" display="LV_ALIGN_DEFAULT" /> means <ApiLink name="lv_align_t" display="LV_ALIGN_TOP_LEFT" /> with LTR base direction and <ApiLink name="lv_align_t" display="LV_ALIGN_TOP_RIGHT" /> with RTL base direction. Default: <ApiLink name="lv_align_t" display="LV_ALIGN_DEFAULT" />, inherited: No, layout: Yes, ext. draw: No.
