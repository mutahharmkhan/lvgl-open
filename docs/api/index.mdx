LVGL (Light and Versatile Graphics Library) is an open-source embedded graphics library providing everything you need to create beautiful UIs for any MCU, MPU, or display type.

<ApiSummary isIndex functions={3348} enums={176} structs={315} typedefs={362} macros={2432} variables={118} />

### Navigating the API

The API is organized into **modules** that mirror the LVGL source tree:

- **Core** — Base object, events, display, input device, and style system
- **Widgets** — All built-in UI widgets (buttons, labels, sliders, charts, etc.)
- **Draw** — Rendering backends (SW, OpenGL, Vulkan, SDL) and draw primitives
- **Drivers** — Display and input device driver interfaces
- **Layouts** — Flex and grid layout engines
- **Font** — Font rendering and management
- **Libs** — Integrated third-party libraries (PNG, GIF, FreeType, etc.)
- **Others** — Utilities like timers, file system, and snapshot

Each module page lists its headers with per-file API counts. Click any header to see its full function, enum, struct, and macro documentation.
