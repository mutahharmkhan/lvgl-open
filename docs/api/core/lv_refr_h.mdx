### lv_refr_now

Redraw the invalidated areas now. Normally the redrawing is periodically executed in <ApiLink name="lv_timer_handler" /> but a long blocking process can prevent the call of <ApiLink name="lv_timer_handler" />. In this case if the GUI is updated in the process (e.g. progress bar) this function can be called when the screen should be updated.

```c
void lv_refr_now(lv_display_t *disp)
```

<span className="sr-only">Parameters</span>

| Name | Type | Description |
|------|------|-------------|
| `disp` | <ApiLink name="lv_display_t" display="lv_display_t *" /> | pointer to display to refresh. NULL to refresh all displays. |


</ApiMember>
