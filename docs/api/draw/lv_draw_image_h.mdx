| `skew_x` | <ApiLink name="int32_t" /> | Parallelogram like transformation of the image horizontally in 0.1 degree unit. E.g. 456 means 45.6°. |

| `blend_mode` | <ApiLink name="lv_blend_mode_t" /> | Describes how to blend the pixels of the image to the background. See <ApiLink name="lv_blend_mode_t" /> for more details. |
| `antialias` | <ApiLink name="uint16_t" /> | 1: perform the transformation with anti-aliasing |
| `tile` | <ApiLink name="uint16_t" /> | If the image is smaller than the image_area field of <ApiLink name="lv_draw_image_dsc_t" /> tile the image (repeat is both horizontally and vertically) to fill the image_area area |

| `sup` | <ApiLink name="lv_draw_image_sup_t" display="lv_draw_image_sup_t *" /> | Used internally to store some information about the palette or the color of A8 images |
| `image_area` | <ApiLink name="lv_area_t" /> | Used to indicate the entire original, non-clipped area where the image is to be drawn. This is important for:
