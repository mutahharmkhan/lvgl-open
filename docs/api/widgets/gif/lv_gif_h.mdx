### lv_gif_set_color_format

Set the color format of the internally allocated framebuffer that the gif will be decoded to. The default is LV_COLOR_FORMAT_ARGB8888. Call this before <ApiLink name="lv_gif_set_src" /> to avoid reallocating the framebuffer.

```c
void lv_gif_set_color_format(lv_obj_t *obj, lv_color_format_t color_format)
```

<span className="sr-only">Parameters</span>

| Name | Type | Description |
|------|------|-------------|
| `obj` | <ApiLink name="lv_obj_t" display="lv_obj_t *" /> | pointer to a gif object |
| `color_format` | <ApiLink name="lv_color_format_t" /> | the color format of the gif framebuffer |


</ApiMember>
